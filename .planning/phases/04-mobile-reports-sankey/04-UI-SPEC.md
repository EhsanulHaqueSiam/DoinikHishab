# Phase 4 -- UI Design Contract

> Visual and interaction contract for the Mobile Reports and Sankey phase. This phase replaces the existing View-based report charts with victory-native interactive charts, adds swipe navigation and tap-to-filter, introduces a Financial Health tab with Age of Money and Days of Buffering trend charts, and delivers a Sankey cash flow visualization with responsive layout (vertical on mobile, horizontal on web) and privacy mode. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, chart axis tick spacing |
| sm | 8px | Compact element spacing, legend dot gaps, tooltip inner padding |
| md | 16px | Default element spacing, card padding, chart container padding |
| lg | 24px | Section padding, chart-to-legend gap, Sankey node vertical spacing |
| xl | 32px | Layout gaps, time range pill strip horizontal padding |
| 2xl | 48px | Major section breaks, Sankey diagram top/bottom margin |
| 3xl | 64px | Page-level spacing |

Exceptions:
- Chart touch target minimum: 44px x 44px for tappable bar/segment areas (Apple HIG minimum).
- Time range pill height: 36px (not on 8-point grid but satisfies touch target minimum with 8px vertical padding around 20px text).
- Sankey node minimum width: 20px on mobile, 12px on web (proportional to data, minimum ensures tappability).

**Source:** Phase 3 UI-SPEC spacing scale (unchanged base), new exceptions for chart interaction targets.

## Typography

| Role | Size | Weight | Line Height | Tailwind Class | Phase 4 Usage |
|------|------|--------|-------------|----------------|---------------|
| Body | 14px | 400 (regular) | 1.5 | `text-sm` | Chart tooltips body text, legend descriptions, Sankey node labels |
| Label | 10px | 700 (bold) | 1.4 | `text-2xs font-bold` | Chart axis labels, time range pill labels, section headers ("SPENDING", "FINANCIAL HEALTH"), privacy mode toggle label |
| Heading | 18px | 700 (bold) | 1.2 | `text-lg font-bold` | Report section titles, Sankey header, tooltip amount values |
| Display | 20px | 700 (bold) | 1.2 | `text-xl font-bold` | Total spending amount, net worth hero amount, AoM current value in trend chart |

**Bengali text note:** Same as Phase 3 -- SpaceMono lacks Bengali glyphs. System falls back to device default. Bengali strings approximately 1.3x longer than English. All chart labels and tooltips must use flex layouts, not fixed widths. Sankey node labels truncate with ellipsis when node width is insufficient.

**Source:** Phase 3 UI-SPEC typography (consistent), tailwind.config.js fontSize tokens.

## Visual Hierarchy

### Reports Screen (top to bottom)

1. **Report Type Tabs** -- Horizontal tab strip, scrollable if needed. `bg-surface-100 border-b border-border`. Active tab: `border-primary-600` underline + `text-primary-700`. Sticks to top.

3. **Period Label** -- Current period text centered. `text-sm font-bold text-foreground mb-2`. Shows "March 2026" (or range for 3M/6M/1Y/All). Visual weight: low.
4. **Chart Card** -- Card wrapper containing the active chart. `mx-4`. Visual weight: maximum (primary content).
5. **Chart Legend** -- Below chart within card. Colored dots + labels. Visual weight: low-medium.
6. **Summary Row** -- Totals/comparison below chart. Visual weight: medium.
7. **Bottom Spacer** -- `h-20` for tab bar clearance.

1. TimeRangeSelector + Period Label

1. TimeRangeSelector + Period Label

## Interaction Contracts

### Time Range Pill Tap

- **Tap**: Sets active time range, reloads chart data for that range
- **Touch target**: Each pill minimum 44px wide, 36px tall (padding ensures this)
- **Feedback**: `active:bg-surface-400` on inactive pills, `active:bg-primary-400` on active pill

### Swipe Month Navigation

- **Gesture**: Horizontal pan via react-native-gesture-handler PanGesture
- **Minimum displacement**: 50px horizontal to trigger month change
- **Velocity threshold**: 500px/s -- fast swipes trigger regardless of displacement
- **Animation**: Slide transition with `withTiming`, 250ms duration, `Easing.out(Easing.cubic)`
- **Boundary behavior**: Subtle bounce-back (no rubber-band on RN, handled by clamp)
- **Period label update**: Immediate on gesture end, before animation completes

### Chart Bar/Segment Tap

- **Tap**: Highlights tapped element, dims others, shows ChartTooltip
- **Touch target**: Minimum 44px height for each bar/segment (bar height 24px + 8px gap + 12px padding = 44px)
- **Feedback**: Tapped bar scales to 1.02x via `withSpring`, 150ms
- **Dismiss**: Tap anywhere else on chart or 3-second auto-dismiss timeout

### Sankey Node Tap

- **Tap**: Highlights node and connected links. Non-connected links dim to 10% opacity. Shows tooltip with node total.
- **Touch target**: Node rectangle minimum 20px width (enforced in layout). Pressable overlay with `hitSlop={4}`.
- **Feedback**: Node border stroke changes from 0px to 2px in `text-foreground` color.
- **Dismiss**: Tap anywhere else or 3-second auto-dismiss.

### Privacy Toggle

- **Tap**: Toggles privacy mode on/off. Persists to MMKV.
- **Touch target**: 36px x 36px Pressable wrapping 20px icon
- **Feedback**: `active:opacity-60`. Icon swaps between Eye and EyeOff.
- **Animation**: Labels crossfade between amount and percentage with `withTiming`, 200ms opacity transition.

### Report Tab Tap

- **Tap**: Switches active report. Chart content swaps immediately (no animation between tabs).
- **Touch target**: Full tab area, minimum 48px height
- **Feedback**: `active:bg-surface-200` Pressable
