## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, dot indicators on calendar days |
| sm | 8px | Compact element spacing, calendar cell internal padding |
| md | 16px | Default element spacing, card padding, section gaps |
| lg | 24px | Section padding, calendar grid outer margins |
| xl | 32px | Layout gaps between major sections (summary card to calendar) |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions: Calendar day cells use 44px minimum touch target (accessibility). Bottom sheet content uses 20px horizontal padding (matching BalanceCard pattern p-5).

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px (text-sm) | 400 (regular) | 1.5 |
| Label | 10px (text-2xs) | 700 (bold) | 1.4 |
| Heading | 20px (text-xl) | 700 (bold) | 1.2 |
| Display | 36px (text-hero) | 700 (bold) | 1.17 (42px line-height, matching BalanceCard) |

Notes:
- Labels use uppercase + tracking-widest for section headers and status badges (established pattern from BalanceCard, TimeRangeSelector)
- Body text for list items uses text-sm font-bold for primary labels, text-2xs text-surface-800 for secondary info
- Burn rate totals use text-hero for visual impact on subscription summary

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #070b16 (background) | Screen background, deep void |
| Secondary (30%) | #111827 (card) / #0c1021 (surface-100) | Cards, calendar grid background, list rows, bottom sheet |
| Accent (10%) | #0d9488 (primary teal) | Paid status dots, active tab indicator, primary CTA, chart line for positive balance |
| Destructive | #f87171 (danger) | Overdue status, danger zone fill below zero line, remove subscription action |

Accent reserved for:
- Paid status dot on calendar (teal dot)
- Paid status badge in list view (Badge variant="default" = teal)
- "Add Subscription" primary CTA button
- Forecast chart line when balance is above zero
- Active view toggle pill (Calendar/List)

Additional semantic colors for this phase:

| Color | Token | Usage |
|-------|-------|-------|
| Saffron | #e6a444 (accent) | Upcoming status dot and badge — matches existing accent color |
| Red | #f87171 (danger) | Overdue status dot and badge, danger zone area fill, delete action |
| Teal | #0d9488 (primary) | Paid status dot and badge |
| Gray dashed | #3b4d63 (surface-600) | Zero reference line on forecast chart |
| Red fill | rgba(248, 113, 113, 0.15) (glow.danger) | Semi-transparent area fill below zero in forecast |

Status-to-Badge mapping (reuse existing Badge component):

| Status | Badge Variant | Dot Color Class |
|--------|--------------|----------------|
| Paid | default (teal) | bg-primary-500 |
| Upcoming | warning (amber/saffron) | bg-accent |
| Overdue | danger (red) | bg-danger |

## Interaction Contracts

### Calendar Grid

| Interaction | Behavior |
|-------------|----------|
| Tap a day with bills | Open BillDaySheet bottom sheet showing that day's bills |
| Tap left arrow | Navigate to previous month with slide-left animation |
| Tap right arrow | Navigate to next month with slide-right animation |
| Tap a day without bills | No action (no haptic, no sheet) |

### View Toggle (Calendar / List)

| Interaction | Behavior |
|-------------|----------|
| Tap Calendar pill | Switch to BillCalendarGrid view |
| Tap List pill | Switch to BillListView |
| Default state | Calendar view |

### Subscription Management

| Interaction | Behavior |
|-------------|----------|
| Tap "Confirm" on suggestion | Subscription moves to confirmed list, card animates out. Haptic feedback (light) |
| Tap "Not a subscription" | Suggestion card dismisses with fade-out. Stored in MMKV dismissed list so it does not reappear |
| Swipe left on subscription row | Reveal red delete zone (80px wide, bg-danger, trash icon white). Release to confirm removal |
| Tap "Add Subscription" | Open AddSubscriptionForm bottom sheet |
| Submit form | Validate payee (required) and amount (> 0). Save to MMKV. Close sheet. Haptic feedback (success) |

### Cash Flow Chart

| Interaction | Behavior |
|-------------|----------|
| Tap data point | Show ChartTooltip with date + projected balance amount. Auto-dismiss after 3 seconds |
| Toggle 30/60/90 | Recalculate projection with new horizon. Chart re-renders with animation |
| Danger zone | Semi-transparent red fill (glow.danger token) below zero line. Zero line as gray dashed (surface-600) |

## Animation Contract

| Animation | Library | Duration | Easing |
|-----------|---------|----------|--------|
| Calendar month transition | react-native-reanimated | 300ms | Easing.out(Easing.ease) |
| Subscription card confirm (animate out) | react-native-reanimated | 250ms | Easing.in(Easing.ease) — slide up + fade |

| Swipe-to-delete reveal | react-native-gesture-handler | Follows finger | Spring (damping: 20, stiffness: 200) |

| Chart render | react-native-gifted-charts built-in | 500ms | Default easing |

## Accessibility Contract

| Element | Requirement |
|---------|------------|
| Calendar day cells | accessibilityRole="button", accessibilityLabel="[Day number], [N] bills due", accessibilityState selected for today |
| Calendar dots | Decorative only (no separate a11y label), info conveyed via parent cell label |
| Status badges | accessibilityLabel includes status text (e.g., "Paid", "Overdue") |
| Swipe-to-delete | accessibilityActions with "delete" action as alternative to swipe gesture |
| View toggle pills | accessibilityRole="tab", accessibilityState selected |
| Forecast range pills | accessibilityRole="button", accessibilityState selected |
| Chart data points | accessibilityLabel with date + amount on each interactive point |
| Bottom sheets | accessibilityViewIsModal=true when open |
| Empty states | accessibilityLiveRegion="polite" |
| Burn rate totals | accessibilityLabel="Monthly subscription total: [amount]" |
